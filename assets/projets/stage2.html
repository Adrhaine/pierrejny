Stage 2ème Année à La Poste

Description : Lors de ce stage j'ai eu comme mission de développer une application web pour sécuriser l'étage via une lecture de QR code et de code NFC. Pour cela j'ai développer une application en Vue.js en utilisant une API FastAPI et une base de données MariaDB. J'ai alors appris Vue.js et ayant vu les principes d'une API dans le BTS SIO j'ai rapidement pu m'adapter. Cette sécurisation ce fait via une Raspberry PI pour lire les QR codes et les codes NFC. Tout le développement de mon application s'est fait sur une Machine Virtuelle Ubuntu sur VirtualBox. J'ai travaillé en mode "DevOps" avec les méthodes agiles adaptées en faisant des sprints avec des réunions hebdomadaires. De plus, l'infrastructure était constitué d'un serveur Apache pour mettre l'application sur le web, AWS et Gitea. Mes différentes documentations durant le stage ont été faites sur Obsidian, un logiciel de prise de notes en markdown très utile.

Fonctionnalités principales :

  • Fonctionnalité 1 : Développement d'un Application web avec Vue.js
  • Fonctionnalité 2 : Développement d'une API FastAPI
  • Fonctionnalité 3 : Sécurisation du site de La Poste
